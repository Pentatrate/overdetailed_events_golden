# Overdetailed Events Golden

(EA Mod)
by Random22 & Flecha Da Foxy & Pentatrate

## Info

More up to date info about this mod can be found in the [beatblock modding discord server (invite)](https://discord.gg/VDvPUSCdGZ)
It has a dedicated post named `Overdetailed Events Golden` in the `ea-mods` forum

(And is also availiable for the Demo version of the game! Check out the post in `demo-mods` with its original name, `Overdetailed Events`)

## How to install

1. Download zip:
	Either directly download the repository as a zip

	Then rename the mod folder to `overdetailed_events_golden`

	![Screenshot downloading as zip](https://github.com/user-attachments/assets/0653e3ef-cfe0-4b41-825f-a7e786feda4d)

	or download the "latest" release (The latest release may not have features added in later commits) (**I recommend this**)

	![Screenshot to find releases](https://github.com/user-attachments/assets/2acbead3-fad3-476b-9525-43fb1d728cb6)

2. Unzip and place in `beatblock/Mods/` folder (where all the other mods are)

	**!!!Reminder!!!**: Make sure the mod.json file is in `beatblock/Mods/overdetailed_events_golden/mod.json` and **NOT** `beatblock/Mods/overdetailed_events_golden/overdetailed_events_golden/mod.json`

3. Relaunch modded beatblock
